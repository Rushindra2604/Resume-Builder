# AI Resume Builder

## 🚀 Live Demo
https://resume-builder-rushindras-projects.vercel.app/

## 📌 About
An AI-powered resume builder that generates professional resumes using Gemini API.

## ✨ Features
- Generate resumes using AI
- Editable sections
- Clean UI
- Download resume

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js
- Gemini API

## ⚙️ Setup
1. Clone repo
2. npm install
3. node server.js

⚠️ Note: Backend is currently running locally. Frontend is deployed on Vercel.
